# Tictactoe

Implémentation C++/JavaScript du jeu de Tictactoe.


- Nom: ACHEHBOUN
- Prénom: ALADINE
- Groupe: G1


## Compilation du projet C++

Projet CMake classique, dans le dossier `cpp` :

```
nix-shell
mkdir build
cmake ..
make
...
```

## Compilation du projet JavaScript

Dans le dossier `js` :


```
nix-shell
make
make run
...
```
